# auctor

Auctor is a simple static site generator.

## Getting started

To keep it simple, auctor expects the following structure:

- _includes
  - (Any EJS includes)
- _layout
  - default.ejs
  - (Any other EJS layout files)
- site
  - assets (CSS, JS, images)
  - content
    - (Any EJS, MD, or directories)
